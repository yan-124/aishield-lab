# 3D模型替换记录

## 任务概述
将网站首页Hero区域的SVG盾牌图标替换为3D模型展示。

## 修改内容

### 文件修改
- **文件**: `src/components/HeroSection.tsx`
- **修改类型**: 替换HeroVisual组件

### 具体变更

#### 1. 导入新增
```typescript
import { useMemo } from 'react'
import { Canvas, useFrame } from '@react-three/fiber'
import * as THREE from 'three'
```

#### 2. 新增3D相关组件
- `loadGLB()`: 加载GLB格式3D模型
- `SceneLighting()`: 3D场景光照设置
- `GlowRing()`: 发光圆环装饰
- `Particles()`: 粒子效果
- `LoadingSpinner()`: 加载动画
- `AnimatedModel()`: 模型动画（旋转+浮动）
- `Hero3DModel()`: 主3D模型组件

#### 3. HeroVisual组件重写
- 移除了原有的SVG盾牌代码
- 添加了基于@react-three/fiber的3D Canvas
- 使用shieldy-a.glb作为默认模型
- 保留了浮动动画效果

### 使用的3D模型
- **路径**: `/shieldy-3d/shieldy-a.glb`
- **备用模型**: shieldy-b.glb, shieldy-c.glb, shieldy-d.glb
